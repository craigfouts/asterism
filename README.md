Asterism - Experiments with Point Cloud Clustering
==================================================
Implementations of semantic segmentation algorithms and spatial topic models applied to point cloud data. Current methods include variations of latent Dirichlet allocation, a vector quantized-autoencoder, and a neural clustering process. Also a test bed for ATLAS, an adaptable topic model for latent areal structures.
